중첩 조건문

조건문의 축약

논리 연산자

        A && B  - 앞 뒤의 조건이 모두 참이면, 결과로 true를 반환
        A || B - 앞 뒤의 조건이 둘 중 하나라도 참이면 결과로 true를 반환

switch

        switch(조건) {
            case a : 조건의 결과가 a일 때 할 일;
                break; -> 조건이 a에서 참이면 break로 문이 종료

            case b : 조건의 결과가 b일 때 할 일;
                break;
        }

for 반복문

        초기문 : 초깃값 설정 (1번으로 실행)
        조건문 : 언제까지 반복할지 조건 설정 (2번으로 실행)
        증감문 : 반복문을 수행하고 초기값을 늘리거나 줄이는 문장을 설정

        for(초기문; 조건문; 증감문) {
            반복될 구문;
        }

        순서는 초기문 -> 조건문 -> 반복될 구문 -> 증감문 이후에는 (조건문 -> 반복될 구문 -> 증감식)이 반복된다

while 반복문

        초기문;
        while(조건문) {
            //반복할 코드
            증감식
        }

do while 반복문

        초기문;
        do {
            //일단 반복할 일
            //증감 여부에 대해서 작성
        } while (조건);
        초기문을 먼저 기술하고 do 구문안의 코드를 최소 한번 실행
        조건에 맞을 때 까지 반복

forEach 반복문

        배열의 원소에서만 사용 가능 / item은 배열의 각 요소
        index는 배열의 각 요소의 인덱스 번호 / all은 배열의 전체의 값

        배열.forEach(function(item,index,all) {
            //반복할 일을 작성한다.
        })

for in 반복문

        객체의 값들을 가지고 반복
        객체에서만 사용 가능 / item은 객체의 각 property
        for (item in 객체) {
            item들 마다 반복할 일을 작성
        }

for of 반복문

        for(item of 배열) {
            item마다 반복할 일을 작성해준다.
        }
