Array

    배열은 비슷한 성격의 여러 데이터들을 한 저장소에 저장해 놓고,
         필요할 때 데이터를 꺼내서 사용할 때 생성합니다.

    빈 배열 만드는 방법
         var 이름 = new Array();
         var 이름 = [];

    값을 지정하는 건 배열명 [0] = 값;

    값을 확인하는 방법은 배열명[1];(저장해 둔 후 그 안의 값을 꺼내는 방법)
        배열의 개수 = 배열명.length;

join

    배열의 값을 하나의 문자열로 합쳐준다.(join);
        사용법 - 배열명.join('구분자'); //구분자는 배열을 구분해서 나열할 때 쓸 기호이다.

배열 원소의 정렬

        순서를 반대 또는 오름차순, 내림차순, 알파벳순, 가나다순으로 정렬
        reverse - 원소의 순서를 반대로 정렬
        sort - 원소의 정렬을 오름차순, 내림차순, 알파벳순, 가나다순 등으로 변경

        기존의 배열에 값을 추가 또는 제거
        push - 기존의 배열에 마지막에 값을 추가
        pop - 기존 배열의 마지막 원소를 제거
        ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
        unshift - 기존 배열의 원소의 가장 앞에 새로운 값을 추가
        shift - 배열의 첫 원소를 제거
        ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
        기존의 배열에 값을 추가 or 제거 (특정 구간에 있는 것들을 추출)
        slice - 기존의 배열을 유지하고 복사해서 가져오는 것 (기존 배열 변경 X)
        splice - 기존 배열에서 지정한 범위의 원소를 오려내어 추출 (원본의 배열이 수정되어 있음)
        ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
        배열의 값들을 합치거나 복사하는 방법
        concat - 배열을 병합함
        filter - 주어진 함수의 조건을 통과하는 모든 원소를 모아 새로운 배열을

reverse

        원소의 정렬을 역순 변경 (배열의 원본이 수정된다.)
        배열명.reverse();

sort

        원소들을 오름차순, 내림차순 정렬 (배열의 원본이 수정된다.)
        배열명.sort();

push

        기존의 배열에 마지막에 값을 추가

unshift

 
    기존 배열의 원소의 가장 앞에 새로운 값을 추가

shift

    배열의 첫 원소를 제거

slice

        slice는 기존에서 복사를 하기 때문에 원본의 값이 변하지 않음

splice


       splice는 기존의 배열에서 오려내서 가져온다.
        
    

    
concat

        값을 병합해준다. 원본.concat(새로운 배열을 넣어주면 원본에 더해줌.)

            조건에 부합하는 원소들을 추출하여 새로운 배열을 생성
