템플릿 리터럴

        변수와 문자열을 연결할 때 유용하게 사용
        `(backtick,백틱)`으로 감싸고 변수명은 ${ }안에 기술

        ex. var msg = '문자' + 변수명1 + 변수명2 + '문자';
        ↓아래와 같이 작성이 가능해졌다.
        ex. var msg = `문자 + ${변수명1 + 변수명2} + 문자`;

가변변수와 불변변수

var vs let
        var는 전역변수와 함수 내에서만 사용할 수 있는 지역변수로 구분
        블록 스코프 밖에서도 변수를 참조할 수 있어 문제가 발생한다.
        스코프는 변수에 접근 가능한 영역을 의미
        let은 블록 스코프의 범위에서만 값을 유지 (스코프내에서만 변수 설정이 가능 중괄호안에서)
    
const
        한번 지정하면 절대 변경되지 않는 불변변수
        var, let은 변수의 값 변경 가능
