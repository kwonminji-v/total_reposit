타이머 함수

setTimeout - 일회성 타이머 설정

        시간을 설정하고, 해당 시간이 소진되면 정해진 할 일을 실행
        할 일 = 함수 = function() { } / 시간 = millisecond 
        setTimeout(해야할 일(function), 시간(time)); ex> 1000 = 1s
        시간을 한번 세팅 후, 소진되면 할일을 한다

setInterval - 반복성 타이머 설정

        설정한 시간동안 반복적으로 정해진 작업을 수행
        setInterval(할 일(function), 시간)
