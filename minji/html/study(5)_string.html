String

charAt();

        charAt - 문자열에서 특정 인덱스 번호에 해당하는 문자를 반환
        배열명.charAt(인덱스 번호);
        문자열도 배열과 마찬가지로 문자 하나하나를 배열로 이용할 수 있다.

IndexOf, Search

        indexOf - 문자열에서 특정 문자의 인덱스 번호를 반환 / 일치하는 값이 없다면 -1을 반환
        문자열.indexOf('찾을 문자')
        대상 문자열에 특정 문자열이 존재하는지 확인할 때 유용하다.

concat

        concat - 문자열과 문자열을 병합
        문자열.concat(문자열);
        대상.concat(대상1)

replace

        replace - 문자열과 특정 문자를 변경 / a문자열에서 b를 C로 변경
        a.replace('b','c') / 새로운 변수에 변경한 것을 저장할 수 있음
        검색된 문자열이 여럿 존재할 경우, 첫번째로 검색된 문자열만 치환
        대상.replace('바꾸고자 하는 타겟문자,문자열', 값);

slice

        slice - 인덱스 번호에 해당하는 문자열을 복사하여 추출 / 시작 번호부터 끝 번호 엎 부분까지 추출(-1)
        문자열.slice(시작번호,끝번호) ex) hello world -> st.slicce(0,5); -> hello까지만 출력된다.
                                  012345678910 
        대상.slice(start,end) end의 바로 전 번호까지만 복사해준다.

split

        분리되어 있지 않은 문자열에서 특정 문자열의 패턴을 이용하여 배열로 생성
        var org = 'httm/css/javascript; -> org.split('/')
        대상.split('구분자') -> 무엇으로 구분지어서 새로운 배열을 만들것인가
