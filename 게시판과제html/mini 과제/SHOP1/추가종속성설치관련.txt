5.폴더구조를 적는다. 
(폴더구조, 

eslint와 관련 플러그인들
설명: JavaScript 코드의 문법 오류나 스타일 가이드를 따르지 않은 코드를 찾아주는 도구입니다.
설치: npm install eslint eslint-plugin-react
prettier
설명: 코드 포맷터로, 코드를 일관된 스타일로 유지하는 데 도움이 됩니다.
설치: npm install prettier
husky와 lint-staged
설명: Git 커밋 전에 린팅이나 테스트를 실행할 수 있도록 도와주는 도구입니다.
설치: npm install husky lint-staged
typescript
설명: 자바스크립트에 정적 타입을 추가하는 프로그래밍 언어로, 코드의 안정성을 높여줍니다.
설치: npm install typescript
jest
설명: 자바스크립트 코드 테스트를 위한 테스트 프레임워크입니다.
설치: npm install jest
